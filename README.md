# Rule-Based AI Chatbot

## Overview

The Rule-Based AI Chatbot is a Python application that simulates human-like conversations using predefined rules and pattern matching. The chatbot accepts user input, analyzes keywords, and generates appropriate responses based on programmed logic.

This project demonstrates the fundamentals of Artificial Intelligence, Natural Language Processing (NLP), and conversational systems through a simple rule-based approach.

---

## Features

- Interactive text-based conversation
- Greeting and farewell responses
- Date and time information
- Programming-related queries
- Motivational messages
- Joke responses
- User-friendly interface
- Rule-based decision making

---

## Technologies Used

- Python 3

---

## How It Works

1. The user enters a message.
2. The chatbot analyzes the input.
3. Keywords are matched with predefined rules.
4. A suitable response is generated and displayed.
5. The conversation continues until the user exits.

---

## Example Conversation

You: Hello

Bot: Hi there! How can I help you today?

You: What is Python?

Bot: Python is a beginner-friendly programming language used in AI, web development, and data science.

You: Tell me a joke

Bot: Why do programmers prefer dark mode? Because light attracts bugs!

You: Bye

Bot: Goodbye! Have a wonderful day.
